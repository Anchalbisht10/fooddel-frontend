
# Food Delivery Frontend

This repository contains the frontend code for the Food Delivery project, built with React.js and Vite. 
It provides the user interface for browsing the menu, adding items to the cart, signing up/in, and placing orders. 
It communicates with the deployed backend APIs to fetch data and manage cart and order operations.


The frontend is deployed and can be accessed at this live link: [https://fooddel-frontend-pi.vercel.app/].
which is connected to the backend [https://fooddel-backend-5yfj.onrender.com/]


To run locally, clone this repository, install dependencies with `npm install`, create a `.env` 
file with the backend live URL (e.g., `VITE_API_URL=<Your Backend Live Link>`), 
then start the development server with `npm run dev`. For production, build the project using `npm run build` and serve it via your preferred hosting service. 


This repository works with the Food Delivery backend repository [https://github.com/Anchalbisht10/Food-delivery], which handles all API operations, authentication, and database management. Both repositories together form the complete Food Delivery project.
